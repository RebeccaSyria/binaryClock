# BinaryClock 
An in-progress command line binary clock written in C 
## Usage: 
- compile: `make` 
- run: `./binaryClock [arguments]`
	- Arguments: 
		- `-c [0..7]`	Set color 
		- `-h`		View help
		- `-s`		Toggle seconds
		- `-l`		Toggle labels
		- `-b`		Toggle bold font 
		- `-d`		Toggle decimal view 
		- `-o [char]`	Set on character (default: 1)
		- `-f [char]`	Set off character (default: 0)
